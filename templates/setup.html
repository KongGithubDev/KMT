ไม่ต้องสร้าง Google Cloud Project แค่คัดลอก headers จาก browser

⚠️ สำคัญ: ใช้ request จาก /browse เท่านั้น

อย่าใช้ /account_menu หรือ /get_search_suggestions - จะไม่มี cookie ที่ถูกต้อง

ขั้นตอนการรับ Headers:
  1. เปิด music.youtube.com และเข้าสู่ระบบ
  2. กด F12 เปิด Developer Tools → แท็บ Network
  3. Filter พิมพ์: /browse (สำคัญ!)
  4. คลิกเข้า playlist ใดก็ได้ หรือกดเล่นเพลง
  5. หา request ที่มี /browse ใน URL
    ตัวอย่าง: https://music.youtube.com/youtubei/v1/browse?...
  6. คลิกขวาที่ request → Copy as cURL (bash)
  7. วางด้านล่างนี้ แล้วกด เชื่อมต่อบัญชี

วางคำสั่ง cURL ที่คัดลอกมาทั้งหมด
